# AI Policy Assistant

An AI-powered policy assistant built with Streamlit, ChromaDB, and LLMs.

## Features
- Ask questions about company policies
- Retrieval-Augmented Generation (RAG)
- ChromaDB vector database
- Streamlit web interface
- Fast and accurate policy search

## Tech Stack
- Python
- Streamlit
- ChromaDB
- OpenAI / LLMs
